استانداردهای کد نویسی

JavaScript ES6+

  • استفاده از Class-Based Components
  • Async/Await برای عملیات شبکه
  • Event Delegation و Handlerها
  • Modular Import/Export
class UserManager {
constructor() {
this.token = null;
this.init();
}

async login(credentials) {
const response = await fetch('/api/login');
return await response.json();
}
}

PHP OOP

  • Class-Based Architecture
  • PDO برای دیتابیس
  • RESTful API Design
  • Error Handling یکپارچه
class ApiHandler {
public function handleRequest() {
try {
$data = json_decode(file_get_contents('php://input'), true);
$this->process($data);
} catch (Exception $e) {
http_response_code(500);
}
}
}

قراردادهای نام‌گذاری

فایل‌ها

  • 📄 snake_case.html
  • 📄 component_name.js
  • 📄 manager_class.php
  • 📄 utility_functions.css

کلاس‌ها

  • PascalCase برای کلاس‌ها
  • camelCase برای متدها
  • UPPER_CASE برای ثابت‌ها
  • _prefix برای privateها

متغیرها

  • camelCase برای متغیرها
  • descriptive_names
  • is/has/can برای booleanها
  • handler/callback برای functions
